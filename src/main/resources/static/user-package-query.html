用户套餐查询

查询说明

关联关系
  • 直接关联：User → Package（当前套餐）
  • 历史关联：User → UserPackage → Package（套餐历史）
查询方式
  • 当前套餐：GET /api/user-packages/current/{userId}
  • 套餐历史：GET /api/user-packages/history/{userId}
  • 套餐详情：GET /api/user-packages/details/{userId}

数据库查询示例

查询类型 SQL语句 说明
用户当前套餐 SELECT * FROM users u JOIN packages p ON u.package_id = p.id WHERE u.user_id = ? 通过直接关联查询当前套餐
用户套餐历史 SELECT * FROM user_package up JOIN packages p ON up.package_id = p.id WHERE up.user_id = ? 通过中间表查询套餐历史
用户信息+套餐 SELECT u.*, p.* FROM users u LEFT JOIN packages p ON u.package_id = p.id WHERE u.user_id = ? 查询用户信息和关联的套餐
