新注册功能测试

功能更新说明

  • 订阅时长：不再保存到数据库，仅用于前端计算
  • 自动续费：默认不启用，保存到数据库
  • 支付方式：默认年付，保存到数据库
  • 激活时间：注册次日激活，试用期和宽限期累加到日期计算中
  • 价格计算：套餐总价 = 月价格 × 支付方式倍数

数据库字段说明

字段名 类型 默认值 说明
auto_renew BOOLEAN FALSE 自动续费，默认不启用
payment_method VARCHAR(20) 'yearly' 支付方式，默认年付
trial_period INT 0 试用期天数，默认0
grace_period INT 0 宽限期天数，默认0
total_price DECIMAL(10,2) NULL 套餐总价（价格 × 订阅时长）
